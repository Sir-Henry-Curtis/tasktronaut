Welcome to the Tasktronaut documentation. Whether you're just getting started or looking to use structured workflows, this site explains the user-facing extension surface.

## What is Tasktronaut?

Tasktronaut is an AI coding assistant that lives in your editor. It can read and write files, run terminal commands, fetch approved URLs, and help you build features through natural conversation and structured workflows. Consequential actions remain behind approval gates unless you explicitly configure auto-approval.
